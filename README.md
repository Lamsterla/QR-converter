# QR-converter

A lightweight, no-API-key QR code generator built entirely with PHP. Enter any text, link, or contact info and instantly generate a downloadable QR code — no third-party API keys, subscriptions, or external accounts required.

## ✨ Features

- **Instant QR Generation** — Convert any text or URL into a QR code on the fly.
- **No API Key Required** — Works fully self-contained / via a free public QR endpoint, with no signup or key management needed.
- **Simple, Lightweight Setup** — Just PHP, no heavy dependencies or build tools.

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Backend | PHP |
| Output | QR code image (generated/rendered server-side) |

## 🔑 No API Key Needed

Unlike many QR generators that require signing up for a third-party service and managing an API key, this tool generates QR codes without any external key or authentication — making it simple to deploy and use immediately.
